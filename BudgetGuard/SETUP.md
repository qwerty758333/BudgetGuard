# BudgetGuard Setup
 
  ## Prerequisites
  - Node.js 16+ (check: node --version)
  - npm 8+ (check: npm --version)
 
  ## Installation
  1. git clone https://github.com/[org]/budgetguard.git
  2. cd budgetguard
  3. npm install
 
  ## Development
  npm run dev
  Open http://localhost:5173
 
  ## Production Build
  npm run build
  npm run preview
  Open http://localhost:4173
 
  ## Deploy
  - Automatic: Push to main → Netlify auto-deploys
  - Manual: Drag dist/ to Netlify dashboard
